Actividad 1: Instalación y Configuración de Linux (Kubuntu y Rocky Linux)

Documentación del proceso de instalación de las distribuciones asignadas al Grupo 8: Kubuntu (Gráfico) y Rocky Linux (Consola). Se incluye el particionamiento manual y la instalación de herramientas requeridas.

1.1 Instalación en Ambiente Gráfico (Kubuntu)

Particionamiento Manual y Requisitos

Se realizó la instalación completa de Kubuntu en una máquina virtual. El particionamiento manual del disco se configuró con particiones separadas para `/`, `swap` y `/home`. El sistema quedó con el entorno gráfico KDE funcional.

Navegadores y Editores Instalados

Para el entregable, se instalaron al menos dos navegadores web (ej. Firefox, Brave) y dos editores/IDEs de desarrollo (ej. VS Code, PyCharm).

1.2 Instalación en Ambiente Consola (Rocky Linux)

Configuración y Herramientas Básicas

La instalación de Rocky Linux se ejecutó en modo consola (sin interfaz gráfica). Se aplicó el particionamiento manual del disco y se incluyeron herramientas básicas de línea de comandos requeridas.

  • Herramientas básicas instaladas: `make`, `rsync`, `curl`, `git`, `wget`.
  • Configuración de red y conectividad a internet.

Verificación de Conectividad y Herramientas

Se demuestra la correcta instalación y funcionamiento de la conectividad y las herramientas requeridas. 

                    # Comandos de verificación en Rocky Linux
                    curl -V 
                    git --version
                    ping 8.8.8.8 -c 4
