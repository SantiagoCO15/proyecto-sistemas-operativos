Actividad 2: Instalación y Configuración de Editores de Texto

Instalación de dos editores de texto/IDEs con configuración básica personalizada en Kubuntu y Rocky Linux. Se demuestra la activación de colores de sintaxis y números de línea.

2.1 Instalación y Configuración de Editores (Vim y VS Code)

Editor CLI: Vim (Rocky Linux)

Se instaló y configuró Vim en la distribución de consola (Rocky Linux). Se demuestra la configuración del archivo `.vimrc` para activar números de línea y *syntax highlighting*. 

                    # Contenido del archivo .vimrc
                    set nu
                    syntax on
                    set autoindent

Editor Gráfico: Visual Studio Code (Kubuntu)

Se instaló Visual Studio Code en Kubuntu. Se personalizó con *themes* y extensiones básicas para desarrollo de scripts Bash y Dockerfiles.
