Editor CLI: Vim (Rocky Linux)

Se instaló y configuró **Vim** en la distribución de consola (Rocky Linux)[cite: 42]. Se demuestra la configuración del archivo `.vimrc` para activar números de línea y *syntax highlighting*[cite: 48, 49, 54].

                set nu
                syntax on
                set autoindent
