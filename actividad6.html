Actividad 6: Instalación y Configuración de Kubernetes (Minikube)

Instalación de un clúster Kubernetes local usando Minikube. Despliegue de la imagen de Docker (v2.0) a través de un Deployment y exposición del servicio.

6.1 Instalación de Minikube y Configuración de kubectl

Comandos Iniciales y Verificación

[cite_start]Se instaló **Minikube** [cite: 178] [cite_start]en Kubuntu junto con `kubectl`[cite: 180]. [cite_start]El clúster se inició usando `minikube start` [cite: 181] [cite_start]y se verificó su estado[cite: 182]. 

                    # Comandos de inicio y verificación
                    minikube start --driver=docker
                    [cite_start]kubectl cluster-info # [cite: 190] Captura de cluster-info
                    [cite_start]kubectl get nodes # [cite: 191] Listado de nodos

6.2 Creación de Deployment y Exposición de Servicio

Archivo YAML de Deployment y Despliegue

[cite_start]Se creó un Deployment YAML [cite: 198] [cite_start]utilizando la imagen `grupo8/proyecto-so:v2.0` de Docker Hub[cite: 199]. [cite_start]Se aplica el archivo y se verifica el estado de los Pods[cite: 200, 201]. 

                    # [cite_start]Fragmento del Deployment YAML [cite: 205]
                    apiVersion: apps/v1
                    kind: Deployment
                    spec:
                      replicas: 1
                      template:
                        spec:
                          containers:
                          - name: app-container
                            image: grupo8/proyecto-so:v2.0 
                            ports:
                            - containerPort: 8080

Archivo YAML de Service y Acceso

[cite_start]Se expone el servicio mediante un Service YAML [cite: 202] (tipo NodePort o similar) para permitir el acceso. [cite_start]Se demuestra el acceso al servicio desplegado[cite: 203]. 

                    # Comandos de despliegue y verificación
                    kubectl apply -f deployment.yaml
                    kubectl apply -f service.yaml
                    [cite_start]kubectl logs [NOMBRE_DEL_POD] # [cite: 209] Captura de Logs
                    minikube service [NOMBRE_DEL_SERVICIO] --url # Acceso al servicio
